# 🤖 Search Algorithms in Python (DFS, BFS, Minimax, Greedy & A*)

A Python project that implements some of the most important search and decision-making algorithms used in Artificial Intelligence, graph traversal, pathfinding, and game development.

The project demonstrates how different search strategies work and compares their behavior, efficiency, and use cases.

---

## 📚 Algorithms Included

### 🌳 Depth-First Search (DFS)

DFS explores a graph by going as deep as possible before backtracking.

#### Characteristics
- Uses Stack (or Recursion)
- Memory efficient
- Suitable for graph traversal and tree exploration

---

### 🌐 Breadth-First Search (BFS)

BFS explores all neighboring nodes before moving to the next level.

#### Characteristics
- Uses Queue
- Guarantees shortest path in unweighted graphs
- Widely used in graph traversal problems

---

### 🎮 Minimax Algorithm

Minimax is a decision-making algorithm commonly used in two-player games.

#### Characteristics
- Simulates opponent moves
- Maximizes winning chances
- Used in Tic-Tac-Toe, Chess, Checkers, and other games

---

### 💡 Greedy Search

Greedy Search always chooses the locally optimal option at each step.

#### Characteristics
- Fast decision making
- Does not always produce the optimal solution
- Useful in optimization problems

---

### ⭐ A* Search Algorithm

A* combines path cost and heuristic estimates to find optimal paths efficiently.

#### Characteristics
- Uses heuristic functions
- Optimal and complete when using admissible heuristics
- Commonly used in navigation systems and game AI
